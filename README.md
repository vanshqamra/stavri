This repository contains the marketing, lead-generation, and light B2B portal for Rab Noor Pvt Ltd, operating under marblesnearme.com.

## Tech Stack
- Next.js 14 (Pages Router)
- TypeScript
- Tailwind CSS via CDN import

## Available Pages
- Marketing pages: home, about, sourcing-from-india, importing-to-europe, logistics-shipping, for-architects, contact, quote-builder
- Product catalog with detail pages for each marble
- Blog with static articles
- Projects gallery + case study pages
- B2B portal (landing, login placeholder, dashboard mock)
- Company section (about/team/careers)
- Region-specific SEO landing pages under `/regions`

## Forms & APIs
Forms submit to placeholder API routes under `/api/contact`, `/api/quote`, and `/api/quote-request`. Extend these handlers to integrate with your CRM or email service.

## Development
```
npm install
npm run dev
```
